# Toshiba / Viessmann Cjenik

Interni alat za pregled cijena Toshiba proizvoda s Viessmann šiframa i izračunom popusta.


## Funkcionalnost

- Unos jedne ili više Toshiba šifri (svaka u novi red)
- Prikaz Viessmann (VCS) šifre uz svaki proizvod
- Kataloška cijena iz Toshiba partner portala
- Automatski izračun cijene s popustom -52% i -48%
- Oznaka "nema cijene" za proizvode koji postoje u bazi ali bez cijene
- Oznaka "nije u bazi" za šifre koje nisu pronađene
